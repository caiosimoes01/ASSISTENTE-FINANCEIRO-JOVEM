# ASSISTENTE-FINANCEIRO-JOVEM
Uma calculadora financeira clara, confiável e educativa para jovens brasileiros, que simula juros compostos, rendimentos, aportes mensais, inflação real e equivalência de taxas — com precisão monetária de nível profissional.  Objetivo: ser a calculadora financeira mais honesta e simples para quem está aprendendo a investir no Brasil.
